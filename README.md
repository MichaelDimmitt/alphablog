# README
This is the Alpha Blog App built during the Udemy Ultimate Ruby on Rails Developer course

## Developer Installation
1. git clone https://github.com/wbhinton/alphablog.git;
2. bundle install;
3. rake db: create
4. rake db: migrate
5. rails s
